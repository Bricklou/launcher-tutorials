# Guide

## Execution

Pour utiliser l'outil vous devez executer la commande (dans le dossier ou est telecharger le binaire):

```bash
./flowupdater-json-creator{votre version}
```

Si vous etes sous Linux, vous risquez d'avoir une erreur, pour pallier a ce probleme :

```bash
chmod +x ./flowupdater-json-creator{votre version}
```

## Utilisation

Le logiciel offre un outil en ligne de commande intuitif qui vous permet de creer votre JSON en repondant a une suite de questions.
Vous ne devriez pas rencontrer de probleme, si vous en rencontrez, ouvrez une ISSUE sur le repository Github ou mentionnez `Zuygui` sur le discord de Support Launcher.

Happy Hacking !
