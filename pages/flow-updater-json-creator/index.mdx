# Introduction

Pour remplacer l'outil [FlowUpdater Json Creator](https://github.com/FlowArg/FlowUpdaterJsonCreator) ecrit par FlowArg qui n'est plus mise a jour depuis 2020, une version CLI (en ligne de commande) a ete ecrite en Rust pour remplacer cet outil.
Dans ce guide, vous apprendrez a l'utiliser et generer des fichiers JSON compatibles avec la librairies FlowUpdater.

Actuellement, le projet ne prend en charge que le fournisseur de mods CurseForge.
