# Installation

Pour installer le logiciel, vous pouvez le telecharger depuis la page [Release du projet github](https://github.com/zuygui/flowupdater-json-creator/releases) et prendre la version correspondante a votre systeme d'exploitation.

Sinon, vous pouvez installer le projet depuis les sources mais cela requiert [rust](https://rust-lang.com) et cargo.
Pour ceci, executez :

```bash
git clone https://github.com/zuygui/flowupdater-json-creator.git
cd flowupdater-json-creator
cargo build --release
```
